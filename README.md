# Project title

E-commerce Customer Intelligence & Retention Analysis

# Project Overview

This project analyzes customer purchasing behavior, retention patterns, and revenue trends using transactional e-commerce data. The goal is to generate actionable insights that support business growth and customer retention strategies.

# Business Problem

The company wants to understand:
- Which customers generate the most revenue
- How customer retention changes over time
- Which products drive sales growth
- Opportunities to improve customer loyalty

  # Dataset
Single table  that contains :
- product details
- customer details
- sales data


Key Fields:
- Customer ID
- Order Date
- Product Category
- Revenue

  # Tools Used

- SQL
- Excel
- GitHub

  # Data Cleaning

Performed the following:

- Standardized date formats
- Created calculated columns

  # Analysis

- Revenue Analysis
- Customer Segmentation
- Cohort Retention Analysis
- Product Performance Analysis
- KPI Tracking


# Key Insights

- Returning customers contributed 88% of total revenue.
- Electronics generated the highest average order value.
- Customers purchasing within 7 days of signup showed higher retention.

# Recommendations

- Increase focus on customer retention campaigns.
- Promote high-performing product categories.
- Reduce time-to-first-purchase through onboarding incentives.
